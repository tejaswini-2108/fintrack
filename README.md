# FinTrack

## Problem Statement
Managing personal finances manually is difficult and users often fail to detect fraudulent transactions. There is a need for an intelligent system that helps track expenses, analyze spending patterns, and detect fraud.

## Tech Stack
- Frontend: React.js, Tailwind CSS
- Backend: FastAPI
- Database: Supabase PostgreSQL
- AI/ML: Python, Scikit-learn, Pandas
- Deployment: Vercel, Render
